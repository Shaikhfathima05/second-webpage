# HealthCare Clinic Website

A modern, responsive healthcare clinic website built with React and Vite.

## Features

- **Responsive Navigation Bar** - Fixed navbar with smooth scrolling to sections
- **Hero Section** - Eye-catching landing section with call-to-action buttons
- **Services Showcase** - Display of medical services with hover effects
- **Doctors Profile** - Meet the medical team with detailed information
- **Appointment Booking** - Interactive form for scheduling appointments
- **Contact Information** - Complete contact details and working hours
- **Smooth Scrolling** - Enhanced user experience with smooth navigation
- **Scroll to Top Button** - Quick navigation back to the top
- **Mobile Responsive** - Fully optimized for all device sizes

## Technologies Used

- React 19.2.0
- Vite 7.3.1
- CSS3 with modern features
- ESLint for code quality

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The application will open at `http://localhost:5174`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
